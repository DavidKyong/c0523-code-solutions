# express-post-json-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What does the `express.json()` middleware do and when would you need it?
  Built in middleware function in express that parses incoming requests with json payloads and it is based on the body-parser. req.body. You would want to use it when mutiple app.methods are going to have to parse to JSON.
